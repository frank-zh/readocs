================
Sphinx安装
================

本章节以Windows环境为例，介绍如何在安装Sphinx.

安装Python
================

Sphinx的安装依赖Python。在安装Sphinx前，需要先安装Pyhton.


安装Sphinx
================

打开CMD，执行以下命令安装Sphinx:

.. code-block:: bash

		pip install sphinx

然后安装restructuredtext-lint:

.. code-block:: bash

		pip install restructuredtext-lint
