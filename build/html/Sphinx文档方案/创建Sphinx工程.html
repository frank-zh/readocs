3. 创建Sphinx工程¶

本章节介绍如何使用sphinx和rst创建及发布一个文档工程。

3.1. 创建文档工程¶

  1. 在本地新建一个Sphinx项目文件夹。

  2. 打开命令行终端(例如，Windows CMD), 进入项目文件夹。

  3. 输入以下命令，进入项目创建流程。

sphinx-quickstart

  1. 参考以下内容，创建项目:

  1. 进入工程文件目录,然后进入Source文件夹。该文件夹包含几个文件。其中Index.rst用于设置文档TOC， conf是Sphinx的配置文件，可以配置输出文档的主题。可以新建.rst文件，编辑文档内容。

具体文档编写语法，参考以下链接内容：

  • sphinx对rst文档的帮助文档

  • rst的官文帮助文档

3.2. 设置主题¶

Sphinx提供默认主题。如果需要下载更多主题，参考: 官方内置主题

此处以sphinx_rtd_theme主题为例进行安装。进入Source目录，执行下面命令：

pip install sphinx_rtd_theme

安装完毕后，修改conf.py文件，保存。

html_theme = 'sphinx_rtd_theme'

3.3. 生成文档¶

在工程目录（非source目录），执行make命令：

make html

3.4. 预览文档¶

生成的htm目录在工程的build/html目录下。双击html目录下的index.html完成预览。
